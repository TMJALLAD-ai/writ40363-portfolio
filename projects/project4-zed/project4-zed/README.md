# Budgeting Web Application

This project is a budgeting web application that allows users to manage their finances effectively. It includes features for adding, updating, and deleting budget items, as well as visualizing spending through a graph.

## Project Structure

```
project4-zed
├── src
│   ├── index.html          # Main HTML document for the web application
│   ├── css
│   │   └── styles.css      # Styles for the web application
│   ├── js
│   │   ├── app.js          # Main JavaScript file that initializes the application
│   │   ├── budget.js       # Functions and classes related to budgeting
│   │   ├── visualization.js # Data visualization for budgeting data
│   │   └── storage.js      # Handles data storage
│   └── assets
│       └── fonts           # Directory for font files
├── package.json            # Configuration file for npm
└── README.md               # Documentation for the project
```

## Features

- **Budget Management**: Add, update, and delete budget items.
- **Data Visualization**: View spending patterns through interactive charts and graphs.
- **Data Storage**: Save and retrieve budget data using local storage.

## Getting Started

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd project4-zed
   ```

3. Install dependencies:
   ```
   npm install
   ```

4. Open `src/index.html` in your web browser to view the application.

## Usage

- Use the application to track your income and expenses.
- Visualize your spending habits with the provided charts.
- Customize the styles in `src/css/styles.css` to fit your preferences.
